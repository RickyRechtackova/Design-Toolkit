Color Palette Generator

A color palette generator is a tool designed to assist designers in creating cohesive and visually appealing color schemes for their projects. Using principles of color theory, the generator generates a set of colors that complement each other well and form a harmonious palette. After colors are generated, user can click on them and the color code will automatically copy into their clipboard.
